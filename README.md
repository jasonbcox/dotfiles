# Jason's dotfiles
------------------

## Setup

1. Run initial setup:
```
./setup/setup.sh
```

2. On OSX, make sure the terminal properly sends the following:
  1. ^Up    : \033[1;5A (ESCAPE [1;5A)
  1. ^Down  : \033[1;5B (ESCAPE [1;5B)
  1. ^Left  : \033[1;5D (ESCAPE [1;5D)
  1. ^Right : \033[1;5C (ESCAPE [1;5C)
